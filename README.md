# Hiderxy
Hiderxy é um script AHK para esconder programas e abas, principalmente para executores do Roblox.
## Utilização
No Hiderxy, há 5 slots para esconder os programas (abas). Para esconder um programa em um slot, deve-se utilizar o atalho CTRL junto com o número do slot, por exemplo: 1. Após isso, utiliza-se a tecla H para esconder e a tecla J para mostrar. Lembrando que se deve utilizar o atalho antes de cada ação de esconder ou mostrar. Caso o usuário queira mostrar todos os slots de uma vez, basta utilizar o atalho CTRL + ALT + A.
